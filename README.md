Тема проекта: Онлайн "калькулятор по загразнянию окружающей среды"


Идеи: 
- Список того, чем можно загрязнять среду (Бутылки, пластиковая посуда и тп.);
- Кнопки для того, чтобы указывать кол-во предметов из списка;
- Итоговый подсчет вашего участия в загрязнении планеты таким образом;

Библиотеки:
- Jinja
- Flask
